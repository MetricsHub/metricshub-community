# Getting Help

Use the `-h` or `--help` option to list all the available options:

 ###### Example:

```batch
$ java -jar ${project.artifactId}-[project.version}.jar -h
```

or

```batch
$ java -jar ${project.artifactId}-[project.version}.jar --help
```
